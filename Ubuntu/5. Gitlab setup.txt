From https://about.gitlab.com/downloads/#ubuntu1604.

Instead of using Postfix you can also use Sendmail or configure a custom SMTP server and configure it as an SMTP server (https://gitlab.com/gitlab-org/omnibus-gitlab/blob/master/doc/settings/smtp.md).

sudo apt-get install curl openssh-server ca-certificates postfix

curl -sS https://packages.gitlab.com/install/repositories/gitlab/gitlab-ce/script.deb.sh | sudo bash
sudo apt-get install gitlab-ce

The default account's username is root.
