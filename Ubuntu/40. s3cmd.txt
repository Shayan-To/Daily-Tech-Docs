# Config

`~/.s3cfg`:

```
[default]
access_key = 
secret_key = 
host_base = https://shayan-to-ivzxs.hs3.ir
host_bucket = https://shayan-to-ivzxs.hs3.ir
use_https = True
```

# Upload large files

    s3cmd --no-check-md5 --verbose --multipart-chunk-size-mb=100 --continue-put put <FILE> s3://st-temp

Then immediately Ctrl+C to get the upload id. Then:

    while true; do s3cmd --no-check-md5 --verbose --multipart-chunk-size-mb=100 --continue-put --upload-id=<UPLOAD_ID> put FILE s3://st-temp && break; sleep 3s; done
